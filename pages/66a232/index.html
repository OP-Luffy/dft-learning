安装ASE

# 安装ASE

总结：根据python版本选择ase版本。python3及以上最新即可。python2选3.17.0及以下。学校集群由于其他依赖包版本低，需要装3.9.1。

# Requirements

最新版本的ase需要的环境条件：

  • Python (opens new window) 3.6 or newer
  • NumPy (opens new window) 1.11 or newer (base N-dimensional array package)
  • SciPy (opens new window) 0.18 or newer (library for scientific computing)

Optional but strongly recommended:

  • Matplotlib (opens new window) 2.0.0 or newer for plotting
  • tkinter (opens new window) for ase.gui (opens new window)

Optional:

  • Flask (opens new window) for ase.db (opens new window) web-interface
  • pytest (opens new window) 3.6.1 or newer for running tests
  • pytest-xdist (opens new window) 1.22.1 or newer for running tests in parallel
  • spglib (opens new window) for certain symmetry-related features

# Windows

# 1.安装并配置Python环境

python38即可

# 2. 安装ASE

pip install --upgrade --user ase
1

会安装最新版本的ASE。

# 3. 添加环境变量

将以下添加到PATH
%AppData%\Roaming\Python\Python38\Scripts
1
2

该变量可能不一样，需要自行确定

# Linux

需要注意的是，支持Python2的ase最高版本是3.17.0.学校的集群是python2.7，需要

pip install --user ase==3.17.0
1

该版本要求numpy-1.9.0，而学校配置的1.7.1，需要

pip install --user numpy==1.9.0
1

经测试会报错，没找到原因，把ase降级到3.9.1，正常运行。

pip install --user ase==3.9.1
1

或：下载： https://files.pythonhosted.org/packages/c1/32/c7a102936e877b43a90e3242c95a719c2056fd3184b33767948c16defd22/ase-3.9.1.tar.gz ，上传，解压缩，pip install --user ase-3.9.1/setup.py(绝对路径)。

与新版本不同，此时必须通过ase-gui，而新版本可以ase gui，因为旧版本/home/2020223055092/.local/bin/下没有ase.

# 参考文献

  • https://wiki.fysik.dtu.dk/ase/install.html#installation-using-pip (opens new window)
  • https://pypi.org/project/ase/3.17.0/ (opens new window)

在Windows 10 20H2 版本上，ASE可能报错，可能是由于numpy与Windows 20H2不兼容导致，参考 numpy-in-win20H2.md 。
