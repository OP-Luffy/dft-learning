自旋电荷密度

# 自旋电荷密度

本页以O2的自选电荷密度为例。

# 步骤

本页操作在完成 O2结构优化、自洽、非自洽 中结构优化、静电自洽后进行。

在2-scf/文件夹中进行。

有两种方法获得其自旋电荷密度：

# 方法1：通过VTST脚本

# Usage
$ chgsplit.pl <CHGCAR>

# This example
[zjb@op 2-scf]$ chgsplit.pl CHGCAR
