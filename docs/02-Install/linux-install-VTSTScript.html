配置VTST的脚本

下载

cd ~/app
$ wget https://github.com/northword/Computation-Simulation/raw/main/app/vtstscripts.tgz

解压缩

tar -xzvf vtstscripts.tgz & rm vtstscripts.tar
cd vtstscripts-933
pwd
# 将返回值复制

加入bashrc

vi ~/.bashrc
# 将上面pwd的返回值加入PATH
