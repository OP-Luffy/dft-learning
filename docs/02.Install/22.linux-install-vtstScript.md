# 配置VTST的脚本

## 下载

```bash
$ cd ~/app
$ wget https://github.com/northword/Computation-Simulation/raw/main/app/vtstscripts.tgz
```

## 解压缩

```bash
$ tar -xzvf vtstscripts.tgz 
$ rm vtstscripts.tgz
$ cd vtstscripts-933
$ pwd
# 将返回值复制
```

## 加入bashrc

```bash
vi ~/.bashrc
# 将上面pwd的返回值加入PATH
```
