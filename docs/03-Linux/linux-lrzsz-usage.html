使用方法

上传（rz）

$ rz
  • 上传可以直接拖动，上传文件至当前目录；
  • 同时也可以命令：输入rz回车后，会出现文件选择对话框，选择需要上传文件，一次可以指定多个文件，上传到服务器的路径为当前执行rz命令的目录

下载（sz）

#下载一个文件
$ sz filename 

#下载多个文件
$ sz filename1 filename2

#下载dir目录下所有文件，不包含dir下的文件夹
$ sz dir/*
