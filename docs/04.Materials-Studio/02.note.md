# 一些乱七八糟的为什么

## 为什么要建立超胞？

1. 避免周期性镜像间的互相影响
2. 考虑多分⼦吸附时的多种构象
   
> 如果只计算体性质，用原胞（primitivecell）就可以，建立超胞（supercell）的目的是什么？
> 
> 我们一般建立晶胞，设置k点进行计算，那么建立超晶胞的目的何在呢？
> 
> （1）掺杂问题
> 
> 1)建立超胞是为了获得合适的掺杂浓度；
> 
> 2)清晰地分成置换固溶体或间隙固溶体
> 
> （2）表面吸附问题
> 
> 1）可以模拟不同晶向的吸附密度；
> 
> 2）要避免吸附的气体分子和镜像分子间有作用；
>
